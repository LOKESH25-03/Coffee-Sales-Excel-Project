 # Coffee Sales Dashboard - Excel Business Intelligence

A production-ready Excel dashboard for comprehensive coffee sales analysis featuring:

✅ Interactive timeline controls (2019-2022 data)
✅ Multi-dimensional filtering (Roast type, Size, Loyalty status)
✅ 4 coffee variety trend analysis (Arabica, Excelsa, Liberica, Robusta)
✅ Geographic sales distribution (US, Ireland, UK)
✅ Top customer segmentation and rankings
✅ Real-time chart updates with dynamic slicers

Perfect for:
- Business analysts building portfolio projects
- Coffee shop owners tracking performance
- Excel enthusiasts learning advanced dashboards
- Retail analytics and market analysis

Built with: Excel Pivot Tables | Timeline Controls | Dynamic Charts | Interactive Slicers
